# 📝 To-Do List App

A sleek, simple, and responsive To-Do List app built with HTML and CSS. Perfect for keeping your tasks organized!

✨ Features

- **Add Tasks:** Type your task and hit "Add."
- **Complete Tasks:** Click on a task to mark it as done.
